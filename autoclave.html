Autoclave Text Module

Operate

Pre-Loading Preparation: Inspect chamber for cleanliness and absence of debris. Check water reservoir level (if applicable). Pre-Loading සූදානම: Chamber පිරිසිදුද, කඩදාසි/සුන්බුන් නොමැතිද පරීක්ෂා කරන්න. (අදාළ නම්) water reservoir මට්ටම බලන්න. Pre-Loading தயாரிப்பு: Chamber சுத்தமாகவும் debris இன்றி உள்ளதா பாருங்கள். (தேவையெனில்) water reservoir அளவைச் சரிபார்க்கவும்.

Loading Items: Wear heat-resistant gloves and PPE. Arrange packs to allow steam circulation (do not overload). Ensure instruments are clean, dried, and packaged correctly. Loading: Heat-resistant gloves සහ PPE ඇඳින්න. Steam circulation සඳහා packs අතර ඉඩ තිබෙන ලෙස සකස් කරන්න. Instruments පිරිසිදු, වියළි, නිවැරදිව package කර ඇතැයි බලන්න. Loading: Heat-resistant gloves மற்றும் PPE அணியவும். Steam circulation க்கு இடமளிக்க packs இடைவெளியுடன் அமைக்கவும். Instruments சுத்தம், உலர்வு, சரியான packaging உறுதி செய்யவும்.

Select Cycle: Choose appropriate cycle (e.g., 121 °C for 30 min or 134 °C for 4–5 min). Close and lock the autoclave door securely. Cycle තේරීම: item අනුව cycle තෝරන්න (121°C මිනිත්තු 30 හෝ 134°C මිනිත්තු 4–5). Door වසා lock කරන්න. Cycle தேர்வு: பொருட்களுக்கு ஏற்ப cycle ஐத் தேர்ந்தெடுக்கவும் (121°C 30 நிமிடம் அல்லது 134°C 4–5 நிமிடம்). Door ஐ மூடி lock செய்யவும்.

Start Cycle: Press start; monitor cycle parameters on display. Start Cycle: Start ඔබා cycle parameters display මත බලන්න. Start Cycle: Start ஐ அழுத்தி display இல் cycle parameters கண்காணிக்கவும்.

End of Cycle: Wait until pressure returns to zero before opening door slightly. Allow load to cool before handling. Cycle අවසානය: Pressure zero වන තෙක් බලා සිටින්න; දොර ටිකක් විවෘත කර steam නිදහස් කරන්න. Load සිසිල්වූ පසු පමණක් ගන්න. Cycle முடிவில்: Pressure zero ஆகும் வரை காத்திருங்கள்; door ஐ சிறிது திறந்து steam வெளியிடவும். Load குளிர்ந்த பின் மட்டுமே கையாளவும்.

Extra: Always follow manufacturer’s recommended cycle. Never open the door under pressure. අමතර: නිෂ්පාදක cycle අනුගමනය කරන්න. Chamber පීඩනය යටතේ door විවෘත නොකරන්න. கூடுதல்: உற்பத்தியாளர் பரிந்துரைக்கும் cycle பின்பற்றவும். Chamber அழுத்தத்தில் இருக்கும் போது door திறக்க வேண்டாம்.

Safety Checks (Daily)

Before use: Check pressure gauge (should read zero). භාවිතයට පෙර: Pressure gauge zero දක්වයිද බලන්න. பயன்பாட்டிற்கு முன்: Pressure gauge zero ஆக இருக்க வேண்டும்.

Preventive Maintenance

Daily: Clean chamber and trays. Check water reservoir. දිනපතා: Chamber සහ trays පිරිසිදු කරන්න. Water reservoir පරීක්ෂා කරන්න. தினசரி: Chamber, trays சுத்தம் செய்யவும். Water reservoir சரிபார்க்கவும்.

Troubleshoot

Cycle incomplete → Power failure or low water → restore power, refill, restart. Cycle අසම්පූර්ණ → Power failure හෝ low water → power ලබා, reservoir පුරවා, cycle නැවත ආරම්භ කරන්න. Cycle முழுமையில்லை → Power failure அல்லது low water → power மீட்டெடுத்து reservoir நிரப்பி restart செய்யவும்.

Quick Checklist

Operate: Clean chamber → Load → Select cycle → Start → Zero pressure → Cool. Operate: Chamber පිරිසිදු → Load → Cycle තෝරන්න → Start → Pressure zero → Cool. Operate: Chamber clean → Load → Cycle தேர்வு → Start → Zero pressure → Cool.
